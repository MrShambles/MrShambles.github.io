# Dynamic web pages with JavaScript

<details>
<summary>What are variables in JavaScript?</summary>

In JavaScript, variables are containers for storing data values. They are used to represent and manipulate data within a program. When you declare a variable in JavaScript, you reserve a space in memory to store the value associated with that variable. Variables can hold various types of data, such as numbers, strings, arrays, objects, functions, etc.

</details>

<details>
<summary> What does it mean to "declare" a variable?</summary>

To declare a variable means to create a named storage location in a computer's memory where you can store data.

</details>

<details>
<summary> What is an "assignment" operator, and what does it do?</summary>


An "assignment" operator is a fundamental concept in programming languages, including JavaScript. It is used to assign a value to a variable or a property. The assignment operator is represented by the equals sign ('=')

</details>

<details>
<summary>What is information received from the user called?</summary>


In JavaScript, information received from the user is often referred to as "user input." It can be gather using various methods such as, using the 'prompt()' function, input elements in HTML forms and listening for user events.

</details>
